Esta pagina web contiene contenido para el aprendizaje y desarrollo de actividades sobre los temas de etiquetas semánticas, listas y enlaces. A continuación encontrara una breve descripción de los modulos:

Etiquetas semanticas

Sirven para dividir un documento en partes logicas, indicando el tipo de contenido que se incluye en cada parte representan por tanto secciones lógicas o componentes de unaaplicació Web o un Documento.

Listas

Las listas se utilizan para agrupar información parecida o relacionada en una WEB.

Enlaces

En este mismo contenido se puede agregar varios enlaces como por ejemplo a videos audio o imagenes.
